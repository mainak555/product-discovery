Product Context:

Contoso IT Solutions is building a self-hosted PHP-based website solution for schools.

The product will allow schools to:

* Download and deploy the application on standard hosting (Apache, PHP, MySQL)
* Manage website content through an admin panel
* Display dynamic content such as notices, gallery albums, carousel images, and informational pages

Target Users:

* School administrators with minimal technical expertise

Core Principle:

* Simplicity over complexity
* Usability over feature richness
* Quick deployment and easy maintenance

Objective of Current Phase:

* Define clear and measurable OKRs (Objectives and Key Results)
* These OKRs will later drive feature definition, user stories, and acceptance criteria
